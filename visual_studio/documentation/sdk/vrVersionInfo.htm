Description

    Use the vrGetVersion method to query the library about it's current version number and other version information.

Notes

    None.

Example Code

	vrVersionInfo info;
	vrGetVersion(&info);

	dc << "Product: " << info.product << "\n"; dc << "Vendor: " << info.vendor << "\n"; dc << "Version Num: " << info.majorNum << "." << info.minorNum << "." << info.releaseNum << "\n"; dc << "Build Date: " << info.buildDate << " " info.buildTime << "\n"; // OUTPUT Product: VRaniML Vendor: Great Hill Corporation Version Num: 0.9.3 Build Date: Nov 18, 1998 10:38am
