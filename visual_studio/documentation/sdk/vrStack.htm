Description

    A simple stack class built on top of the vrList. This class is a template class so stacks of any type of object can be built.

Notes

  • None.

Example Code

	// A simple stack of integers.
	vrStack<SFInt32> intStack;
	intStack.Push(1);
	intStack.Push(2);
	intStack.Push(3);

	while (intStack.Peek())
		printf("%d ", intStack.Pop());

	// OUTPUT: 3 2 1
