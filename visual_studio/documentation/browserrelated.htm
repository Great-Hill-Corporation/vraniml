VRaniML VRML 2.0 Node Class Library
         Browser Related Classes

Classes implementing the execution environment of VRML.  This includes maintaining an internal clock, parsing VRML files, maintaining the VRML scene graph and routing and responding to user and system generated events.

There are three classes in this section of the library, but they are three very important classes.  The vrBrowser class is by far the most important implementing all the functionality needed to achieve the VRML execution model.  The vrEvent and vrRoute classes are supporting classes storing information about events and routes respectively.
