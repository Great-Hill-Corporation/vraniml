Step 5: Opening a VRML File		 [Prev] [Next] [Tutorial3]

In this step we will read an actual VRML file. This is a very simple step.

  1. Add the following code to the OnNewDocument member of the CSample3Doc class: 
BOOL CSample3Doc::OnNewDocument()
{
	// Call the base class's OnNewDocument function
	if (!CDocument::OnNewDocument())
		return FALSE;

	// Read the file (note: we hardcode the filename here for ease of instruction).
	if (browser)
		browser->ReadURL("[install_dir]/browser/logo.wrl");

	return TRUE;
}

Where '[install_dir]' is the directory into which you installed the VRaniML™ library (c:/program files/great hill corporation/vraniml by default)..

Recompile and run the program. You will still see the square box when the program starts, but if you take the File|New menu choice you should see the VRaniML logo file. Notice however that it is not being animated.

In the next step we will animate this scene.
