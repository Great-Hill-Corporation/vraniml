Step 1: Getting Started		 [Next] [Tutorial3]

The first step in building a VRaniML™ program based on MFC is to start with a new MFC project.

To do this complete the following steps from the MS Visual C++ IDE (Note: these instructions assume that you are using version 5.0 of Visual C++.):

  1. From the 'File' menu choose 'New'


  3. From the 'Projects' tab choose 'MFC AppWizard (exe)'


Name your program sample3.

Create a single document interface program with all the defaults.


  • Compile and run the program

    • You should see a single MFC window with a white background and all MFC menus and toolbars. Kill this program.

    In the next step we will link to the VRaniML™ library and prepare for reading a VRML 2.0 file.
