Step 3: Adding a Scene Graph		 [Prev] [Next] [Tutorial3]

In this step we will add a vrBrowser to the Document class of our application.

Follow these instructions to complete this step:

  1. Add a vrBrowser data member to the CSample3Doc class. Make sure this is a public data member (to make following steps easier):
vrBrowser *browser;
  • Initialize the new data member in the CSample3Doc constructor. We will set the browser pointer to NULL here. In the next step we will create the actual browser and associate it with the CView class.
    CSample3Doc::CSample3Doc()
{
	browser = NULL;
}
  • To be complete, we want to properly cleanup the browser in the destructor of the Document class (note: deleting the browser will delete the entire scene graph it contains as well as any vrTraversers):
    CSample3Doc::~CSample3Doc()
{
	if (browser)
		delete browser;
}
  • Recompile and run the program. It should behave exactly as it did in Step 2. Kill the program.

    • We now have a browser pointer but it will not render anything (we have not created the scene graph yet, nor have we associated the browser with a window). We will handle both of these things in the next tutorial step.
