This tutorial has taught you how to create a simple VRaniML™ program using MFC.

We discussed:

How to get started.
How to link to the VRaniML™ library, include the header files and .DLLs that are needed to run VRaniML™ programs.
How to add a browser member to the Document class.
How to add a some nodes to the browser's scene graph and render the scene graph.
How to open a VRML file.
How to animate a scene graph.

Of course, the program can be extended in many ways (and in fact was in the Sample Browser):.

Add menu, mouse and keyboard support
Add better support for opening VRML 2.0 .wrl files
Add support for exporting VRML 2.0 .wrl files
Make more interesting objects and animations
Add sound and videos
A million other things limited only by your imagination
