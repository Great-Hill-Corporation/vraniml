        Step 2: Linking in the Library

In this step we will change the project's build settings to link the VRaniML™ library. At the same time we will add the VRaniML™ header file directory to the include search path and copy the VRaniML™ DLLs to your program directory. Finally, we will add the vraniml.h file to our project.

  • Open the Project Settings dialog box

on the C++ Page [Preprocessor Category]:

add '[install_dir]/sdk/include' to the list of additional include directories (where '[install_dir]' is the directory into which you installed the VRaniML™ library (c:/program files/great hill corporation/vraniml by default)).

on the Link Page [Input Category]:

add '[install_dir]/sdk/lib' to the list of additional library directories

  • Close the Project Settings dialog box
  • Copy [install_dir]/runtime/*.dll to your program directory
  • Add the following line to the file sample1.cpp:
    • #include "vraniml.h"
  • Recompile and run the program.

This program should behave exactly as it did in Step 1.  In Step 3 we will actually open a VRML file.
