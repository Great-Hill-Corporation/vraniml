VRaniML Tutorials
        Step 1: Getting Started

The first step in building a VRaniML™ console based program is to start with a new project.  To do this complete the following steps from the MS Visual C++ IDE.  Note: These instructions assume that you are using version 5.0 of MS Visual C++.

  • From the 'File' menu choose 'New'.
  • From the 'Projects' tab choose 'Win32 Console Application'.  Name your new program sample1.
  • From the 'File' menu choose 'New'.
  • From the 'Files' tab choose 'C++ Source File'.   Call this new file sample1.cpp.
  • Add the following code to the file sample1.cpp
    • #include "stdlib.h"
#include "stdio.h"

void main(int argc, char *argv[])
{
	printf("VRaniML Pretty Printer\n");
	printf("Version 1.0\n\n");
}
  • Compile and run this program

You should see the text "VRaniML Pretty Printer" show up on the DOS window.   In the next step we will link to the VRaniML™ library and prepare for reading a VRML 2.0 file.
