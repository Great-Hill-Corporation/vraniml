        Step 3: Opening a VRML File

In this step we create a vrBrowser class and read in a VRML file.

  • Add a vrBrowser variable to the function main.
void main(int argc, char *argv[])
{
	printf("VRaniML Pretty Printer\n");
	printf("Version 1.0\n\n");

	vrBrowser browser;
}
  • Open a file.
void main(int argc, char *argv[])
{
	printf("VRaniML Pretty Printer\n");
	printf("Version 1.0\n\n");

	vrBrowser browser;

	// Read the file (note: we hardcode the filename here
	// for ease of instruction).  Note: Sending NULL to ReadURL
	// will cause the browser to read from stdin.
	browser.ReadURL("[install_dir]/sample browser/logo.wrl");
}

Where '[install_dir]' is the directory into which you installed the VRaniML™ library (c:/program files/great hill corporation/vraniml by default).

  • Recompile and run the program.

This program should behave exactly as it did in Step 2.   We now have a full scene graph.  In the next step we will export the file to the stdout stream.
