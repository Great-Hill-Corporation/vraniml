VRaniML Tutorials
        Step 6: Finishing Touches

This tutorial has taught us to create a very simple VRaniML Pretty Printer.

It has demonstrated a number of important concepts:

  • How to link to a VRaniML program and include the vraniml.h file.
  • The vrBrowser class is the top level of the VRML scene graph and provides the ability to read VRML 2.0 files.
  • vrBrowser's are 'traversed' by vrTraverser classes to apply some action to each node in the scene graph. We used a vrWriteTraverser.
  • The vrWriteTraverser is just one example of a vrTraverser derived class. There are a number of other Traverser classes including:
  • vrOGLTraverser -- Draw the scene graph using OpenGL
  • vrD3DTraverser -- Draw the scene graph using D3D (not yet complete)
  • vrValidateTraverser -- Validate the scene graph

Of course, the program can be extended in many ways. For example:.

  • Take command line input for the parameters of the Write Traverser.
  • Apply a ValidateTraverser to the scene graph looking for invalid data.
  • Write your own traverser to do anything you want, for example exporting to other file formats.
