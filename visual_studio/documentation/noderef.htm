VRML97 Node Classes

Each VRML97 Node as defined in the VRML97 specification has a corresponding VRaniML C++ class.  This section of the manual describes each class in detail.  Some classes are derived from base classes which are defined here.   All VRML97 nodes are ultimately derived from the vrNode class.
