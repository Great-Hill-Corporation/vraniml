VRML97 Node Classes

Each VRML97 Node as defined in the VRML97 specification has a corresponding VRaniML C++ class.  This section of the manual describes each class in detail.  Some classes are derived from base classes which are defined here.   All VRML97 nodes are ultimately derived from the vrNode class.

[ Index | Data Types | Defines | Utility Functions ]

Browser Related Classes (VEM)

Classes implementing the execution environment of VRML.  This includes maintaining an internal clock, parsing VRML .wrl files, maintaining the VRML scene graph and routing and responding to user and system generated events.

VRML97 Node Classes

Classes implementing each VRML node as defined in the VRML specification.  Each node is represented with a separate VRaniML C++ class.  A number of additional base class abstract common functionality to ease code re-use.

Traverser Classes

Classes implementing a traversal of the VRML scene graph.  Different Traverser class perform different operations on the scene graph.  One Traverser might write the scene graph out to a file while another might render it to the screen.  Yet another might validate the scene graph or modify it in some way.
