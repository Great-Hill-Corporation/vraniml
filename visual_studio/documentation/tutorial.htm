VRaniML
        Library Tutorials

Following are three tutorials designed to help you understand how to use VRaniML.   The source code for each tutorial is available for download. 

Tutorial Topics

  • Console Application -- Pretty Printer

A stand-alone console application showing how to use VRaniML to read and write VRML files.  This example could be the basis for creating command line VRML generators.

  • Win32 Application -- VRML File Viewer

A simple Win32 application showing how to use VRaniML under traditional windows programming.  This tutorial builds a VRML scene graph programmatically (that is it is not read from a .wrl file).

  • MFC Application -- Simple VRML Browser

The same basic example as the Win32 application but completed using the MFC framework.   This tutorial shows how to integrate VRaniML into MFC applications.
