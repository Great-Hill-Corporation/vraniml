Utility Class Library Reference Manual
         Data Types

This document describes the data types defined in the Utility Library.  These data types are defined to mimic the VRML specification.  This has the effect of making VRaniML related source code look like VRML.  This makes the learning curve for VRaniML shallow.

As in VRML, there are single valued data types and multi valued data types.  The multi valued types are basically arrays of single valued types.  Each data types is full C++ classes with copy constructors and equals operators so they may be treated like any built in data type such as floating point or integer values.  By this we mean that doing arithmetic (such as interpolation) is supported for each of these data types.
