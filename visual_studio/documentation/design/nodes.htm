VRaniML Design Documentation
         VRML97 Node Classes

This component contains the VRML 2.0 nodes.  These classes are intended to behave as they are defined in the VRML 2.0 specification, however some nodes are either not implemented or are only partially implemented.  For more information click here.

Each group of node classes is defined below.

 

Appearance classes: These classes include the Appearance, FontStyle, ImageTexture, Material, MovieTexture, PixelTexture and TextureTransform classes.   These classes effect the visual appearance of any geometry nodes.

Bindable classes: These classes include the Background, Fog, NavigationInfo and Viewpoint nodes.  There is also a base class called Bindable that support common functionality of each of these nodes.  These classes are what VRML 2.0 calls bindable nodes.  These are classes which can only have one active node of the given type active at any given time.  The Browser class supports the binding and un-binding of these bindable nodes.

Common classes: These classes include AudioClip, DirectionalLight, PointLight, Script, Shape, Sound, SpotLight, and WorldInfo.  Also included is a class called Light which supports common functionality in the light nodes.  These nodes support the overall visual/auditory aspects of the VRML 2.0 scene graph.

Geometry classes: These classes include Box, Color, Cone, Coordinate, Cylinder, ElevationGrid, Extrusion, IndexedFaceSet, IndexedLineSet, Normal, PointSet, Sphere, Text and TextureCoordinate.  Also included in this group are the DataSet and GeometryNode classes which provide common code for some of the geometry classes.  The Dataset is the base class for the IndexedFaceSet, IndexedLineSet and other classes.   The GeometryNode class is the base class for all geometric nodes.  The GeometryNode provides support for the use of the Solid Modeling library as the underlying data representation of for all visual objects in the scene graph.

Grouping classes: These classes include the Anchor, Billboard, Collision, Group, Inline, LOD, Switch, and Transform nodes.  These classes implement any node that provides for the addChild, removeChild and children fields in VRML 2.0.

Interpolator classes: These classes include the ColorInterpolator, CoordinateInterpolator, NormalInterpolator, OrientationInterpolator, PositionInterpolator, ScalarInterpolator.  Also included is the Interpolator class (as base class for all Interpolators).  These classes behave as defined in the VRML 2.0 spec.

Sensor classes: These classes include the CylinderSensor, PlaneSensor, ProximitySensor, SphereSensor, TimeSensor, TouchSensor and VisibilitySensor.  Also included are the PointingDeviceSensor (base class for CylinderSensor, PlaneSensor and SphereSensor) and the Sensor classes (base class for all sensors).  These classes behave as defined in the VRML 2.0 spec.
