VRaniML Design Documentation
         Utility Class Library (VRFC)

A collection of supporting classes used by all other components.   Contains things like collection classes, geometry class, runtime typing, etc.  Click here for more information.

 

This component is a collection of supporting routines and classes used by the rest of the SDK.  It is modeled after the MFC (Microsoft Foundation Classes) to provide a shallow learning curve for programmers familiar with that MS Windows based C++ library.  This component contains three sub-groupings of classes as follows:
Collection classes: A group of classes providing standard C++ collection classes.  This component includes the following classes: Array (supports all MF field types), DoublyLinkedList (intrusive and non-intrusive), and Stack
Geometric classes: A group of classes providing support for various geometric related functions.  This component includes the following classes: AxisAlignedBox (for bounding box calculations), Matrix, Plane, Ray, Rect2D, Rotation, Color, Vector2 and Vector3.  These classes support the Node Library as well as the Solid Modeling library.
Utility classes: A group of classes supporting various other functionality.  This component includes the following classes: BaseNode (base class of all node classes - implements reference counting), Dump (support for writing to files), ImageUtils (support for reading texture and other image files), Runtime (support for runtime typing), SoundUtils (support for sound files -- not completed), TimeUtils (support for runtime execution environment - VEM) and WebUtils (support for fetching files over the internet).
