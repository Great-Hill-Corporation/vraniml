VRaniML
         Reference Manuals

VRaniML contains three separate C++ class libraries.  Two of these libraries provide implementations for basic 3D related operations (such as vector and matrix math) and a fundamental geometric data structure (called the half edge).  The third library (the VRML97 Class Library) implements most of the functionality defined in the VRML97 spec.

Each class library is documented in a separate reference manual.

VRML97 Class Library

This collection of C++ classes implements each of the VRML97 nodes, allows for the reading and writing of VRML97 compliant files, implements the runtime environment as defined in VRML97 as well as implementing a number of other VRML related concepts.  The library is based on the Utility and Solid Modeling Libraries below.

Solid Modeling Library

The Solid Modeling Library implements the internal data representation for all geometry related nodes.  The use of this data structure (the half edge) allows us to anticipate useful algorithms such as mesh decimation and boolean operations for future versions of VRaniML.  This library is also stand-alone and may be used for any purpose.

Utility Class Library

A collection of classes implementing functionality such as debugging support, time management, file reading and fetching over the network, etc.  Collection classes such as lists and arrays as well as basic geometry data types such as matrix and vector classes are also supported.  This library may be used stand-alone.
