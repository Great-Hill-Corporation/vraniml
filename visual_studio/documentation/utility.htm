VRaniML
        Utility Class Library Reference Manual

The Utility Library contains base classes, data type definitions, geometry and container classes as well as a few utility functions.  This library is used extensively by the VRML97 Class Library and the Solid Modeling Library. 

Base Classes

    vrBaseNode, vrField, vrRuntimeClass, vrFieldList

Single Valued Data Types

    SFBool, SFInt32, SFFloat, SFVec2f, SFVec3f, SFColor, SFRotation, SFString, SFImage, SFTime

Multi Valued Data Types

    MFInt32, MFFloat, MFVec2f, MFVec3f, MFColor, MFRotation, MFString

Geometry Classes

    vrBoundingBox, vrMatrix, vrPlane, vrRay, vrRect2D

Container Classes

    vrArrayBase, vrArray, vrList, vrRefCountList, vrStack, vrRefCountStack

Other Classes

    vrDumpContext, vrDumpString, vrTime, vrVersionInfo
