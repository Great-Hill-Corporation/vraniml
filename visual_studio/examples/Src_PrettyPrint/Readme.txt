This is the source code for the VRaniML
pretty printer.  You can use this source
code in complience with the non-disclosure
agreement you signed prior to being given
access to this code.

You should be able to build this application
by opening the file PrettyPrint.dsp in this
folder.

To get this to work change the .dat files
to point to .wrl files that are on your machine
and copy the .dat file to the current working
directory.
